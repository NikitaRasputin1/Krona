# KronaFinal11 — Веб-приложение деревообрабатывающего предприятия

## Структура проекта

```
KronaFinal11/
├── index.php          ← главная страница
├── style.css          ← основные стили
├── products-style.css ← стили страницы продукции
├── script.js          ← логика фронтенда
├── connect.php        ← подключение к БД
├── zayavki_patch.sql  ← создание и заполнение БД
├── produkciya.php     ← каталог продукции
├── kalkulator.php     ← калькулятор стоимости
├── dostavka.php       ← информация о доставке
├── delivery.php       ← оформление доставки
├── cart.php           ← корзина
├── add_to_cart.php    ← добавление в корзину
├── checkout.php       ← оформление заказа
├── order_view.php     ← просмотр заказа
├── order_functions.php← вспомогательные функции заказов
├── send_zayavka.php   ← отправка заявки
├── chat.php           ← чат
├── telegram.php       ← уведомления Telegram
├── login.php          ← вход в систему
├── logout.php         ← выход из системы
├── register.php       ← регистрация
├── profile.php        ← личный кабинет клиента
├── employee.php       ← кабинет сотрудника
├── admin.php          ← панель администратора
├── admin_zayavki.php  ← управление заявками
└── uploads/           ← загружаемые файлы (аватары и др.)
```

## Установка (XAMPP / OpenServer)

1. Скопируй папку `KronaFinal11` в `htdocs` (XAMPP) или `www` (OpenServer)
2. Открой phpMyAdmin → вкладка SQL
3. Вставь содержимое файла `zayavki_patch.sql` и нажми **Выполнить**
4. Открой в браузере: [http://localhost/KronaFinal11/](http://localhost/KronaFinal11/)

## Роли пользователей

- **Гость** — просматривает главную страницу, каталог продукции, калькулятор, информацию о доставке
- **Клиент** — личный кабинет, корзина, оформление заказов, отправка заявок, чат
- **Сотрудник** — кабинет сотрудника, работа с заказами и заявками
- **Администратор** — панель управления пользователями, заявками и контентом сайта
