## Установка (XAMPP)

1. Скопируйте папку `KronaFinal11` в `htdocs` (XAMPP)
2. Открой phpMyAdmin → вкладка SQL
3. Вставьте содержимое файла `krona1.sql` и нажмите "Выполнить"
4. Открой в браузере: http://localhost/KronaFinal11/index.php

## Роли пользователей

- **Гость** — просмотр каталога товаров; поиск и фильтрация товаров; просмотр карточки товара.
- **Клиент** — все функции гостя; регистрация и авторизация; добавление товаров в корзину; оформление заказа; выбор способа доставки: доставка курьером по городу Ярославлю или доставка в пункт выдачи заказов (доставка курьером доступна только в пределах города Ярославля); ввод адреса доставки; просмотр истории заказов; отслеживание статуса заказа.
- **Сотрудник - менеджер** — •	управление товарами (добавление, редактирование, удаление); обработка заказов; изменение статуса заказа; взаимодействие с клиентами; подготовка заказов к отправке. 
- **Администратор** — просмотр и анализ отчетов о продажах; контроль финансовых показателей; управление сотрудниками; контроль складских остатков; учет поступления товаров; контроль выполнения заказов.
